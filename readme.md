# 1.ArcGIS API For Javascript基础教程

## 1.1在本实例中，如果想将代码直接运行需注意以下几点：

* 将ArcGIS API离线环境部署到**IIS**中
* **ArcGIS API的版本是3.17**,博客里面的代码不支持4.X的API
* 网页用火狐浏览器运行（不然有部分代码浏览器不兼容）
* ArcGIS Server发布的服务：**服务名称**和**图层顺序一致**（尤其是查询部分，图层顺序一定要一致）
